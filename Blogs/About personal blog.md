### What is this folder?

- A folder that hold all of my condensed learning of a certain topic/technology/process and is compiled into a blog.
### What is consider a blog?

- It should contain my knowledge on the specific matter/topic/technology/process/concept/principle.
- It should have an example to demonstrate that knowledge (github, website).
- I (myself) should be able to fully understand and debate about this topic quite adequately.
- A blog should be its own folder - containing the main blogs, the research knowledge reference and images (if needed), multiple languages version if possible.
