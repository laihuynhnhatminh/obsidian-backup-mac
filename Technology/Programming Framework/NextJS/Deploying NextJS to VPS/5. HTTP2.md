## Prerequisite:
- Nginx version over `1.9.5`
- Enabled HTTPS/SSL (see [[4. Setting HTTPS with certbot]])

## Getting started

- Open nginx site in `sites-enabled` folder

```bash
sudo nano /etc/nginx/sites-enabled/travelbus-nextjs-demo.com
```

- Update the specific line to this

```bash
listen 443 ssl http2;
```

- Recheck nginx config and restart

```bash
sudo nginx -t
sudo systemctl restart nginx
```
