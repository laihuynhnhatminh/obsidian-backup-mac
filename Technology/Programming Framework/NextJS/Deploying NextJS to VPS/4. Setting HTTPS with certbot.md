## Getting started

- Install `snapd`

```bash
sudo apt update
sudo apt install snapd
```

- If you have installed `certbot` before, we need to do a re-installation, first thing to do is to remove it:

```bash
sudo apt-get remove certbot
```

- If this is a new VPS or after you finished the action before. Install `certbot`

```bash
sudo snap install --classic certbot
```

- Set `certbot` command to be used

```bash
sudo ln -s /snap/bin/certbot /usr/bin/certbot
```

- Get `certbot` certification

```bash
sudo certbot --nginx
```
