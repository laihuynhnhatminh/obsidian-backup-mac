## Start (First setup)

- Installing Nginx

```bash
sudo apt-get update && sudo apt-get install nginx
```

- Open port from ubuntu firewall for Nginx

```bash
sudo ufw allow 'Nginx Full'
```

## Config on Ubuntu

- Move to `/etc/nginx`
- Create a new file named with the domain that you want to use. Example `travelbus-nextjs-demo.com` and update it with this

```bash
sudo touch travelbus-nextjs-demo.com

sudo vi travelbus-nextjs-demo.com
or
sudo nano travelbus-nextjs-demo.com

```

- Saving after edit

```vim
ctrl+x
:wq
```

```bash
server {
        listen 80;
        listen [::]:80;

        root /var/www/html;
        index index.html index.htm index.nginx-debian.html;

        server_name travelbus-nextjs-demo.com www.travelbus-nextjs-demo.com;

        location / {
                proxy_pass http://localhost:3000;
                proxy_http_version 1.1;
                proxy_set_header Upgrade $http_upgrade;
                proxy_set_header Connection 'upgrade';
                proxy_set_header Host $host;
                proxy_cache_bypass $http_upgrade;
        }
}
```

- Run this to activate the site

```bash
sudo ln -s /etc/nginx/sites-available/travelbus-nextjs-demo.com /etc/nginx/sites-enabled/
```

- To avoid **hash bucket memory** if we are to add more application to this VPS

```bash
sudo nano /etc/nginx/nginx.conf
```

- Find this line `# server_names_hash_bucket_size 64;`  and remove `#` 
- To check if nginx setting is good to go:

```bash
sudo nginx -t
```

- If everything is ok, run this commend to restart nginx to get it apply all the changes

```bash
sudo systemctl restart nginx
```

- Remember to clean up all of the open port. Your ports should be:

```bash
Status: active

To                         Action      From
--                         ------      ----
22/tcp                     ALLOW       Anywhere                  
Nginx Full                 ALLOW       Anywhere                  
22/tcp (v6)                ALLOW       Anywhere (v6)             
Nginx Full (v6)            ALLOW       Anywhere (v6)    
```
