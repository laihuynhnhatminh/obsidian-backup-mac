## Eslint from 8 to 9 Migration

<font color=EEE8AA>Eslint 8</font> and <font color=EEE8AA>Eslint 9</font> configuration file has different configurations style. We can use <font color=EEE8AA>Eslint 9</font> `useFlatConfig` flag to allow <font color=EEE8AA>Eslint 9</font> to be able to understand the old configs but from <font color=EEE8AA>Eslint 10+</font>, Eslint will be completely migrate to the new config so we should migrate to the new configuration instead.
